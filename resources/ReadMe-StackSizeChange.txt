StackSizeChange　スタック量変更mod


説明
	対象アイテムを64個スタック可能にします。
	設定ファイルを変更する事で、スタック数を自由に変更できます。
	弓のダメージ増加エンチャントの名前を、矢ダメージ増加に変更します。

	また、使用すると別のアイテムになるアイテム（バケツやミルク、スープなど）を使用した際の
	スタックの変更方法を改善します。

		既存：　残りスタックが１の状態で使うと入れ替わる
			例）64個のバケツを連続でミルクにすると、最後が1個と63個に分かれる

		変更後：残りスタックが１の状態で使うと、変更後のスタックアイテムと入れ替わる
			例）64個のバケツを連続でミルクにすると、最後のバケツがあったスロットに64個のミルクが挿入される

バケツ置き換えについて
	Ver-6 から、スタック可能バケツ類を置き換えるか、新規に追加するかを設定可能になりました。
	既存のバケツを書き換えるMODと競合する場合には、書き換え無しにできます。
	また書き換え無しでもバケツをスタックできるように、スタック可能バケツ類を新規アイテムとして追加できます。
	新規に追加した場合、以下のレシピも追加されます。
		空バケツ　⇔　スタックバケツ　　　　　相互変換レシピ
		鉄インゴット６つ　→　スタックバケツ２つ
		スタック牛乳でケーキ作成
		スタック溶岩入りバケツを燃料に　　　　全ての溶岩バケツを消費しますので１個ずつ投入してください。


対象アイテム
	看板、バケツ、ドア、ベッド、ケーキ、トロッコ、ボート、サドル、レコード
	ポーション、キノコスープ、エンダーパール、雪球、卵
	バケツミルク、水バケツ、溶岩バケツ、エンチャントした本


導入方法
	MinecraftForgeが必要です。
	modsフォルダにzipファイルのまま突っ込んでください。


設定ファイル
	〜〜Max				各アイテムの最大スタックサイズ（1〜64）

//1.7以降暫定廃止。
	BucketReplace		true 　既存のバケツを置き換える
						false　置き換えない
	addStackableBucket	true 　スタック可能バケツ類を新規アイテムとして追加する
						false　追加しない
	StackableBucketID	スタック可能バケツ類の基準ID。実際に使用するIDは４つ。
						+0　スタックバケツ（空バケツ）
						+1　スタック水入りバケツ
						+2　スタック溶岩入りバケツ
						+3　スタック牛乳


内部的な話
	既存のバケツ関連アイテムを置き換えます。（Ver-6から設定ファイルで変更可能）
	これらの処理を行うMODとは競合します。
